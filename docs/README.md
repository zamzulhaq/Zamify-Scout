# Zamify Sentinel

**Autonomous Ecommerce Research Agent**

Zamify Sentinel is an advanced AI-driven browser automation tool designed to autonomously research products across major ecommerce platforms (Tokopedia, Shopee, Lazada, TikTok Shop). Built with a modern terminal UI and Playwright.

## Features
- AI-driven product search
- Autonomous scrolling and data extraction
- Multi-platform market analysis
- Terminal UI with rich panels and live status
- Exports to CSV, JSON, and Excel

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Install Playwright browsers: `playwright install chromium`
4. Run: `python main.py`
