# Development Roadmap

This project follows an iterative release cycle, gradually introducing more complex features.

## V1: Foundation & Data Extraction (Current)
- [x] Basic project setup and modular architecture
- [x] Professional Terminal UI (Rich)
- [x] Simple HTML Scraper (Requests + BeautifulSoup)
- [x] Export to CSV/XLSX/JSON
- [x] Basic Logging System

## V2: AI Integration & Intelligence
- [ ] Connect to Google Gemini API
- [ ] AI-driven product categorization
- [ ] Automated price analysis and margin calculation
- [ ] Basic Supplier Finder module (finding similar items on Alibaba/AliExpress)

## V3: Advanced UI & Analytics
- [ ] Web Dashboard UI (Streamlit or simple FastAPI + React)
- [ ] Competitor trend analysis over time (Database integration)
- [ ] Visual charts for pricing history

## V4: Complete Automation Ecosystem
- [ ] Fully autonomous agent capable of scraping multiple sites automatically
- [ ] Voice AI control (Speech-to-text input)
- [ ] Automated notification system for price drops or new competitor products
