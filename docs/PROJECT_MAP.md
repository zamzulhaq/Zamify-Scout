# Project Map

- `core/`: Core engine components (browser orchestration, task routing, AI logic).
- `research/`: Platform-specific scraping agents (Tokopedia, Shopee) and Market Analyzer.
- `ui/`: Terminal user interface components using Rich (banners, progress bars).
- `utils/`: Data cleaning, logging, and export functionalities.
- `docs/`: Project documentation.
- `exports/`: Destination for all output data files.
- `logs/`: Application execution logs.
