# Zamify Sentinel

### Autonomous Ecommerce Research Agent

Zamify Sentinel adalah AI Ecommerce Intelligence Tool berbasis Python + Playwright yang dirancang untuk melakukan:

* competitor research
* browser automation
* product scraping
* price analysis
* ecommerce intelligence

Project ini bekerja seperti AI Agent:

* membuka browser otomatis
* menelusuri website seperti manusia
* mengambil data real
* menganalisa produk kompetitor
